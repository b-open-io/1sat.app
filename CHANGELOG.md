# Changelog

All notable changes to 1Sat Browser will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-23

### Added
- Desktop browser with native WebView (no Chromium)
- Secure Enclave + Touch ID key protection
- File inscription and ordinal minting
- BAP identity management
- Built-in dApp browser with BRC-100 connectivity
- Native file picker for on-chain publishing
- Real-time blockchain sync terminal

### Security
- Private keys never leave the Secure Enclave
- Zero cloud dependency — fully local wallet
